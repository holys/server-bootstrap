# Cecilia for iTerm2

Mainly inspired by OS X default terminal basic theme.
![](Cecilia_theme.png)

## Installation

1. Download Cecilia.itermcolors(or `Save as`).
2. Double-click the downloaded file.
3. Open the iTerm2 preferences menu to load the preset.

![](preset.png)
